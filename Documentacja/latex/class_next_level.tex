\doxysubsection{Detailed Description}
Klasa reprezentująca obiekt przejścia do następnego poziomu w grze. 

Dziedziczy po klasie \doxylink{class_entity}{Entity}. Obiekt ten działa jako punkt kontrolny lub miejsce, które gracz musi osiągnąć, aby przejść na kolejny poziom.
